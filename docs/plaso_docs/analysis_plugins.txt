 Analysis Plugins  
 
 browser_search : Analyze browser search entries from events. 
 chrome_extension : Analysis plugin to gather information about Chrome extensions. 
 bloom : Analysis plugin for looking up hashes in a bloom file. 
 nsrlsvr : Analysis plugin for looking up hashes in nsrlsvr. 
 sessionize : Analysis plugin that labels events by session. 
 tagging : Analysis plugin that labels events according to rules in a tagging file. 
 unique_domains_visited : A plugin to generate a list all domains visited. 
 virustotal : An analysis plugin for looking up hashes in VirusTotal.
