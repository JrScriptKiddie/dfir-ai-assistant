# Конспекты threat-intel отчётов

Каждый отчёт - отдельная папка `<report_id>/`:
- `original.(pdf|html|md)` - оригинал (если позволяет лицензия)
- `article.json` - структурированный конспект (см. docs/llm_wiki.md)
- `LICENSE.txt` - заметка о лицензии/правилах использования
